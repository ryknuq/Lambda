### Getting Started

**Current Status:** The project is under active development. Stability issues may be present. Please check the commit history and the Discord for the latest updates.

#### Prerequisites
*   [e.g., Visual Studio 2018 or later]
*   [e.g., DirectX SDK]

#### Compilation
1.  Clone the repository: `git clone https://github.com/ryknuq/Lambda.git`
2.  Open the solution file (`.sln`) in Visual Studio.
3.  Set the build configuration to `Release` and `x86`. **MUST BE SET TO THIS, OTHERWISE IT WILL NOT BUILD!**
4.  Build the solution (Ctrl+Shift+B).
5.  The resulting `.dll` will be in the `Build` folder directory.
